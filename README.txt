## Etapes d'execution

Installer la base de donnée livres dans /var/www/html/

Pour commencer dans un teminal lancer le serveur local :
cd /var/www/html/
php -S localhost:8000

Pour lancer le projet final sur votre navigateur aller sur le lien :
http://localhost:8000/webbackend/index.php

Dans le TP1, il y avait des petits exercices secondaire, voici les deux liens pour vérifier ces exercices :

http://localhost:8000/webbackend/compteur.php
http://localhost:8000/webbackend/requete.php


##  Dépendances
Le projet utilise les technologies suivantes :
- PHP pour la logique serveur
- PostgreSQL pour la base de données
- AJAX (JavaScript pour la communication asynchrone avec le serveur
- Avoir la base de données livres
